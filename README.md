# issues导出

> AngularJS + gulp

## 一、install

```bash
npm install
```

* 建议执行 `npm install npm -g` 更新 npm 至最新版本
* 建议删除原来项目的 `node_modules` 文件夹，重新执行 `npm install`;
* 为了加快 npm 安装速度可以执行下面的命令，将 npm 源设置为 `taobao`;

```bash
npm config set registry https://registry.npm.taobao.org
```


## 二、执行下面的命令，启动开发环境：

```bash
gulp
```
